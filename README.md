# BoardGo(Front-End)

> ### 소개
>
> #### BoardGo는 보드게임을 즐기는 사람들에게 만남의 기회를 제공하고, 내 위치 기반으로 모임 장소를 추천해주는 서비스 입니다.
>
> 모임 일정 조율, 참가자 관리, 게임 추천 등 다양한 기능을 통해 친구들과의 보드게임 시간을 더 즐겁고 특별하게 만들어 드립니다.
